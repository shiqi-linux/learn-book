# cscope

cscope -Rbq：生成索引文件。

cs find c|d|e|f|g|i|s|t name

| 命令 | 解释 |
| ---- | ---- |
| 0 或 s | 查找本C符号(可以跳过注释) |
| 1 或 g	| 查找本定义 |
| 2 或 d	| 查找本函数调用的函数 |
| 3 或 c	| 查找调用本函数的函数 |
| 4 或 t	| 查找本字符串 |
| 6 或 e	| 查找本egrep模式 |
| 7 或 f	| 查找本文件 |
| 8 或 i	| 查找包含本文件的文件 |
